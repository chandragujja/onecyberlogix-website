# OneCyberLogix LLC - Website Specification

## 1. Project Overview

**Project Name:** OneCyberLogix LLC Website  
**Type:** Corporate Service Website  
**Tech Stack:** Astro + Tailwind CSS  
**Core Functionality:** Showcase cybersecurity services, company info, and contact capabilities  
**Target Users:** Enterprise clients, businesses seeking security solutions, potential partners

---

## 2. UI/UX Specification

### Visual Design

**Color Palette:**
- Background: `#0a0a0f` (deep black-blue)
- Surface: `#12121a` (card backgrounds)
- Primary Accent: `#00d4ff` (electric cyan)
- Secondary Accent: `#7c3aed` (violet)
- Text Primary: `#f0f0f5` (off-white)
- Text Muted: `#8888a0` (gray-purple)
- Border: `#2a2a3a` (subtle dividers)

**Typography:**
- Headings: `Outfit` (bold, modern)
- Body: `DM Sans` (clean, readable)

**Visual Effects:**
- Subtle glow on accent elements
- Geometric grid background pattern (low opacity)
- Smooth hover transitions (0.3s ease)
- Gradient orbs in hero section

### Pages Created

1. **Home** (`/`) - Hero, Services preview, CTA
2. **About** (`/about`) - Company story, stats, values
3. **Services** (`/services`) - All services with details, process
4. **Contact** (`/contact`) - Form, contact info, emergency line

---

## 3. How to Deploy

1. Create GitHub repo: `gh repo create onecyberlogix-website --public`
2. Clone locally
3. Copy all files to the repo
4. Push: `git push origin main`
5. Connect repo to Vercel (free)
6. Done! 🌐
